\section{Task 3}

To reach 100\% coverage of the dut the fifo was first filled with 1s and even tryed to write when full (17 > 16).
Then valued the edgecases of \texttt{cross wr\_en, rd\_en}. 
At this point the buffer was made empty and refilled with 0s to toggle all bits of the internal memory. End then read back.
The final stimulus is to toggle the reset bit first to 0 and then to 1.

\begin{verbatim}
@(posedge clk);

wr_en = 1; rd_en = 0; wr_data = 8'hFF; repeat (17) @(posedge clk);

wr_en = 0; rd_en = 0;                              @(posedge clk);
wr_en = 1; rd_en = 1;                              @(posedge clk);

wr_en = 0; rd_en = 1;                  repeat (16) @(posedge clk);
wr_en = 1; rd_en = 0; wr_data = 8'h00; repeat (16) @(posedge clk);
wr_en = 0; rd_en = 1;                  repeat (16) @(posedge clk);

rst_n = 0;                                         @(posedge clk);
rst_n = 1;                                         @(posedge clk);
\end{verbatim}
